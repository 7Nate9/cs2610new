Taako's Tech Cheat Sheet

Welcome to my tech cheat sheet! Here you'll find a list of helpful git commands, their descriptions, and working examples. You'll also find HTML tags and CSS styles.
Alternatively, see the full documentation for git, the W3Schools HTML documentation, or the W3Schools CSS documentation.

The .hidden CSS class was borrowed from WebAIM's tutorial on creating invisible content that is accessible to screen reader users. This class is used to provide alternative text for CSS background images.

Tech Tables

Git Commands
Command Description Example
git init Creates a git repository in the current directory. git init
git add Displays the sum of all files in the repository. git add
git commit Saves changes to the local repository. (May prompt awkward conversation if repository isn't ready for that kind of thing.) git commit -m "I'm ready to take this to the next level."
git push Pushes saved changes down the tubes to a remote repository. If tubes are clogged, apply pipe snake as needed. git push really hard
git go back The git equivalent of ctrl + Z. Undoes whatever you screwed up. git go back!!1!
git woo Reassures repository after a failed commit. git woo -m "I'm a different person now."
git django Releases a python. A python
git freaky Gives the repository a bottle of bourbon and tells it to have a good time. git freaky, git freaky, everybody in the house git freaky.
git r'lyeh Awakens the Great Old Ones from their eternal slumber. Launches the earth into an unending age of blood and darkness. May corrupt repository. Eternal darnkess
CSS Styles
Style Description Example
display Determines how an element is displayed on the page. .invisible{display:none}
border-style Stands outside the border's dressing room during a shopping montage, giving it advice on which outfit to buy. border-style{Uh-uh. Uh-uh. That's the one.}
WYWIWYG A magical style that allows you to telepathically communicate the style you're going for to the browser, so it can just do what you want it to instead of making you mess around with spacing for hours. WYWIWYG{I just want a simple sidebar, is that too much to ask?!}
HTML Tags
Tag Description Example
<;html>; Contains an HTML document. <;html>;(everything else goes here)<;/html>;
<b> Makes text bold. <b>You said WHAT?!?</b>
<a> Makes an element into the "Alpha" element, giving it superior social status to its peers. <a>I'm the captain now.</a>
<fin> Either terminates the current document, or renders ASCII art of a shark's dorsal fin, depending on the browser. <fin>
